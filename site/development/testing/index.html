    Guia de Testes¶

    Este documento descreve as estratégias, ferramentas e práticas de teste utilizadas no Docker Pilot.

    Visão Geral¶

    O Docker Pilot utiliza uma abordagem abrangente de testes que inclui testes unitários, de integração e end-to-end. Nossa filosofia é "testes como documentação viva" - cada teste deve ser claro, expressivo e serve como documentação do comportamento esperado.

    Estratégia de Testes¶

    Pirâmide de Testes¶

          /\
     /  \  E2E Tests (Poucos)
    /____\
   /      \  Integration Tests (Alguns)
  /________\
 /          \  Unit Tests (Muitos)
/____________\
    • 70% Unit Tests: Testes rápidos e isolados
    • 20% Integration Tests: Testes de componentes integrados
    • 10% E2E Tests: Testes de cenários completos

    Princípios de Teste¶

    1. F.I.R.S.T
    2. Fast: Testes devem ser rápidos
    3. Independent: Testes não devem depender uns dos outros
    4. Repeatable: Resultados consistentes em qualquer ambiente
    5. Self-Validating: Resultado claro (passa ou falha)

    6. Timely: Escritos junto com o código de produção

    7. AAA Pattern

    8. Arrange: Configurar dados e mocks
    9. Act: Executar a operação sendo testada

    10. Assert: Verificar o resultado

    11. Test Naming

    12. Descrever claramente o cenário testado
    13. Usar linguagem de negócio quando apropriado
    14. Formato: should_DoSomething_When_Condition

    Tipos de Teste¶

    1. Testes Unitários¶

    Testam unidades isoladas de código (classes, funções).

    // tests/unit/utils/ValidationUtils.test.ts
describe('ValidationUtils', () => {
  describe('isValidContainerName', () => {
    it('should return true for valid container name', () => {
      // Arrange
      const validName = 'my-app-container';

      // Act
      const result = ValidationUtils.isValidContainerName(validName);

      // Assert
      expect(result).toBe(true);
    });

    it('should return false for name with invalid characters', () => {
      // Arrange
      const invalidName = 'my_app@container!';

      // Act
      const result = ValidationUtils.isValidContainerName(invalidName);

      // Assert
      expect(result).toBe(false);
    });

    it('should return false for empty name', () => {
      // Arrange
      const emptyName = '';

      // Act
      const result = ValidationUtils.isValidContainerName(emptyName);

      // Assert
      expect(result).toBe(false);
    });
  });

  describe('isValidImageTag', () => {
    test.each([
      ['latest', true],
      ['v1.0.0', true],
      ['1.2.3-alpine', true],
      ['', false],
      ['INVALID@TAG', false]
    ])('should validate "%s" as %s', (tag, expected) => {
      expect(ValidationUtils.isValidImageTag(tag)).toBe(expected);
    });
  });
});

    2. Testes de Integração¶

    Testam interação entre componentes.

    // tests/integration/docker/ContainerManager.test.ts
describe('ContainerManager Integration', () => {
  let containerManager: ContainerManager;
  let dockerClient: DockerClient;
  let testContainers: string[] = [];

  beforeAll(async () => {
    dockerClient = new DockerClient();
    containerManager = new ContainerManager(dockerClient);

    // Verificar se Docker está disponível
    const isRunning = await DockerUtils.isDockerRunning();
    if (!isRunning) {
      throw new Error('Docker não está rodando - pule testes de integração');
    }
  });

  afterAll(async () => {
    // Cleanup: remover containers de teste
    for (const containerId of testContainers) {
      try {
        await dockerClient.removeContainer(containerId, { force: true });
      } catch (error) {
        // Ignorar erros de cleanup
      }
    }
  });

  beforeEach(async () => {
    // Garantir que temos uma imagem de teste
    await dockerClient.pullImage('alpine:latest');
  });

  describe('createContainer', () => {
    it('should create container successfully', async () => {
      // Arrange
      const containerName = `test-container-${Date.now()}`;
      const options = {
        image: 'alpine:latest',
        name: containerName,
        cmd: ['sleep', '30']
      };

      // Act
      const containerId = await containerManager.createContainer(options);
      testContainers.push(containerId);

      // Assert
      expect(containerId).toBeTruthy();

      const container = await dockerClient.inspectContainer(containerId);
      expect(container.Name).toBe(`/${containerName}`);
      expect(container.Config.Image).toBe('alpine:latest');
    });

    it('should throw error for invalid image', async () => {
      // Arrange
      const options = {
        image: 'nonexistent:image',
        name: 'test-invalid'
      };

      // Act & Assert
      await expect(containerManager.createContainer(options))
        .rejects
        .toThrow('Image not found');
    });
  });

  describe('listContainers', () => {
    it('should list containers with correct filter', async () => {
      // Arrange
      const testLabel = `test=${Date.now()}`;
      const containerName = `test-list-${Date.now()}`;

      const containerId = await containerManager.createContainer({
        image: 'alpine:latest',
        name: containerName,
        labels: { [testLabel.split('=')[0]]: testLabel.split('=')[1] }
      });
      testContainers.push(containerId);

      // Act
      const containers = await containerManager.listContainers({
        filters: { label: testLabel }
      });

      // Assert
      expect(containers).toHaveLength(1);
      expect(containers[0].Id).toBe(containerId);
    });
  });
});

    3. Testes End-to-End¶

    Testam cenários completos através da CLI.

    // tests/e2e/cli/container-commands.test.ts
describe('Container Commands E2E', () => {
  const CLI_PATH = path.resolve(__dirname, '../../../dist/index.js');
  let testContainers: string[] = [];

  afterAll(async () => {
    // Cleanup containers criados durante os testes
    for (const containerName of testContainers) {
      try {
        execSync(`docker rm -f ${containerName}`, { stdio: 'ignore' });
      } catch {
        // Ignorar erros de cleanup
      }
    }
  });

  describe('list command', () => {
    it('should list containers in table format', () => {
      // Act
      const output = execSync(`node ${CLI_PATH} list containers`, {
        encoding: 'utf8'
      });

      // Assert
      expect(output).toContain('CONTAINER ID');
      expect(output).toContain('IMAGE');
      expect(output).toContain('COMMAND');
      expect(output).toContain('STATUS');
    });

    it('should list containers in JSON format', () => {
      // Act
      const output = execSync(`node ${CLI_PATH} list containers --format json`, {
        encoding: 'utf8'
      });

      // Assert
      expect(() => JSON.parse(output)).not.toThrow();
      const containers = JSON.parse(output);
      expect(Array.isArray(containers)).toBe(true);
    });
  });

  describe('create command', () => {
    it('should create container successfully', () => {
      // Arrange
      const containerName = `test-e2e-${Date.now()}`;
      testContainers.push(containerName);

      // Act
      const output = execSync(
        `node ${CLI_PATH} create ${containerName} alpine:latest --command "sleep 30"`,
        { encoding: 'utf8' }
      );

      // Assert
      expect(output).toContain(`Container ${containerName} created successfully`);

      // Verificar se foi realmente criado
      const inspectOutput = execSync(`docker inspect ${containerName}`, {
        encoding: 'utf8'
      });
      const containerInfo = JSON.parse(inspectOutput)[0];
      expect(containerInfo.Name).toBe(`/${containerName}`);
    });

    it('should show error for invalid image', () => {
      // Arrange
      const containerName = `test-invalid-${Date.now()}`;

      // Act & Assert
      expect(() => {
        execSync(
          `node ${CLI_PATH} create ${containerName} nonexistent:image`,
          { encoding: 'utf8', stdio: 'pipe' }
        );
      }).toThrow();
    });
  });

  describe('interactive mode', () => {
    it('should start interactive menu', (done) => {
      // Este teste requer mocking da entrada do usuário
      const child = spawn('node', [CLI_PATH, 'interactive']);

      let output = '';
      child.stdout.on('data', (data) => {
        output += data.toString();

        if (output.includes('What would you like to do?')) {
          // Simular seleção de 'Exit'
          child.stdin.write('\x1B[B\x1B[B\x1B[B\r'); // Navegar e selecionar
        }
      });

      child.on('close', (code) => {
        expect(code).toBe(0);
        expect(output).toContain('What would you like to do?');
        done();
      });

      setTimeout(() => {
        child.kill();
        done();
      }, 5000); // Timeout de segurança
    });
  });
});

    // tests/helpers/mocks/DockerClient.mock.ts
export class MockDockerClient implements DockerClient {
  private containers: Map<string, Container> = new Map();
  private images: Map<string, Image> = new Map();

  async listContainers(options?: ListOptions): Promise<Container[]> {
    let containers = Array.from(this.containers.values());

    if (options?.filters?.status) {
      containers = containers.filter(c => c.status === options.filters.status);
    }

    return containers;
  }

  async createContainer(options: CreateContainerOptions): Promise<string> {
    const id = `mock-${Date.now()}`;
    const container: Container = {
      id,
      name: options.name || `container-${id}`,
      image: options.image,
      status: 'created',
      created: new Date(),
      ports: options.ports || [],
      volumes: options.volumes || [],
      networks: []
    };

    this.containers.set(id, container);
    return id;
  }

  async startContainer(id: string): Promise<void> {
    const container = this.containers.get(id);
    if (!container) {
      throw new Error(`Container ${id} not found`);
    }

    container.status = 'running';
    container.started = new Date();
  }

  // ... outros métodos

  // Métodos auxiliares para testes
  addMockContainer(container: Container): void {
    this.containers.set(container.id, container);
  }

  clearMockData(): void {
    this.containers.clear();
    this.images.clear();
  }
}

    Factory de Objetos Mock¶

    // tests/helpers/factories/ContainerFactory.ts
export class ContainerFactory {
  static create(overrides: Partial<Container> = {}): Container {
    return {
      id: faker.datatype.uuid(),
      name: faker.random.word(),
      image: 'nginx:latest',
      status: 'running',
      created: faker.date.past(),
      started: faker.date.recent(),
      ports: [],
      volumes: [],
      networks: [],
      ...overrides
    };
  }

  static createRunning(overrides: Partial<Container> = {}): Container {
    return this.create({
      status: 'running',
      started: new Date(),
      ...overrides
    });
  }

  static createStopped(overrides: Partial<Container> = {}): Container {
    return this.create({
      status: 'exited',
      started: undefined,
      ...overrides
    });
  }

  static createList(count: number, overrides: Partial<Container> = {}): Container[] {
    return Array.from({ length: count }, () => this.create(overrides));
  }
}

    Test Helpers¶

    // tests/helpers/TestHelpers.ts
export class TestHelpers {
  static async waitFor(
    condition: () => boolean | Promise<boolean>,
    timeout = 5000,
    interval = 100
  ): Promise<void> {
    const start = Date.now();

    while (Date.now() - start < timeout) {
      if (await condition()) {
        return;
      }
      await new Promise(resolve => setTimeout(resolve, interval));
    }

    throw new Error(`Condition not met within ${timeout}ms`);
  }

  static createTempDir(): string {
    return fs.mkdtempSync(path.join(os.tmpdir(), 'docker-pilot-test-'));
  }

  static async cleanupTempDir(dir: string): Promise<void> {
    await fs.promises.rm(dir, { recursive: true, force: true });
  }

  static randomString(length = 8): string {
    return Math.random().toString(36).substring(2, length + 2);
  }

  static async dockerPull(image: string): Promise<void> {
    try {
      execSync(`docker pull ${image}`, { stdio: 'ignore' });
    } catch (error) {
      throw new Error(`Failed to pull image ${image}: ${error}`);
    }
  }
}

    Custom Matchers¶

    // tests/helpers/CustomMatchers.ts
declare global {
  namespace jest {
    interface Matchers<R> {
      toBeValidContainer(): R;
      toBeRunningContainer(): R;
      toHavePort(port: number): R;
    }
  }
}

expect.extend({
  toBeValidContainer(received: Container) {
    const pass = received.id && received.name && received.image;

    return {
      message: () => `Expected ${received} to be a valid container`,
      pass
    };
  },

  toBeRunningContainer(received: Container) {
    const pass = received.status === 'running';

    return {
      message: () => `Expected container to be running, but status is ${received.status}`,
      pass
    };
  },

  toHavePort(received: Container, port: number) {
    const pass = received.ports.some(p => p.containerPort === port);

    return {
      message: () => `Expected container to have port ${port}`,
      pass
    };
  }
});

    Benchmark Tests¶

    // tests/performance/ContainerOperations.bench.ts
describe('Container Operations Performance', () => {
  let containerManager: ContainerManager;

  beforeAll(async () => {
    containerManager = new ContainerManager(new DockerClient());
  });

  it('should list 1000 containers within acceptable time', async () => {
    // Arrange
    const startTime = Date.now();

    // Act
    await containerManager.listContainers();

    // Assert
    const endTime = Date.now();
    const duration = endTime - startTime;
    expect(duration).toBeLessThan(5000); // menos de 5 segundos
  });

  it('should handle concurrent container creation', async () => {
    // Arrange
    const containerCount = 10;
    const promises: Promise<string>[] = [];

    // Act
    const startTime = Date.now();
    for (let i = 0; i < containerCount; i++) {
      promises.push(containerManager.createContainer({
        image: 'alpine:latest',
        name: `perf-test-${i}`,
        cmd: ['sleep', '1']
      }));
    }

    const containerIds = await Promise.all(promises);
    const endTime = Date.now();

    // Assert
    expect(containerIds).toHaveLength(containerCount);
    expect(endTime - startTime).toBeLessThan(10000); // menos de 10 segundos

    // Cleanup
    await Promise.all(containerIds.map(id =>
      containerManager.removeContainer(id, { force: true })
    ));
  });
});

    Memory Leak Tests¶

    // tests/performance/MemoryLeaks.test.ts
describe('Memory Leak Detection', () => {
  it('should not leak memory during repeated operations', async () => {
    // Arrange
    const initialMemory = process.memoryUsage().heapUsed;
    const containerManager = new ContainerManager(new DockerClient());

    // Act
    for (let i = 0; i < 100; i++) {
      await containerManager.listContainers();

      // Forçar garbage collection se disponível
      if (global.gc) {
        global.gc();
      }
    }

    // Assert
    const finalMemory = process.memoryUsage().heapUsed;
    const memoryIncrease = finalMemory - initialMemory;
    const maxAcceptableIncrease = 10 * 1024 * 1024; // 10MB

    expect(memoryIncrease).toBeLessThan(maxAcceptableIncrease);
  });
});

    Test Runner Scripts¶

    #!/bin/bash
# scripts/test-runner.sh

# Cores para output
RED='\033[0;31m'
GREEN='\033[0;32m'
YELLOW='\033[1;33m'
NC='\033[0m' # No Color

echo -e "${YELLOW}🧪 Running Docker Pilot Test Suite${NC}"

# Verificar se Docker está rodando
if ! docker info > /dev/null 2>&1; then
    echo -e "${RED}❌ Docker is not running. Please start Docker first.${NC}"
    exit 1
fi

echo -e "${GREEN}✅ Docker is running${NC}"

# Executar testes unitários
echo -e "${YELLOW}📋 Running unit tests...${NC}"
npm run test:unit
if [ $? -ne 0 ]; then
    echo -e "${RED}❌ Unit tests failed${NC}"
    exit 1
fi

# Executar testes de integração
echo -e "${YELLOW}🔗 Running integration tests...${NC}"
npm run test:integration
if [ $? -ne 0 ]; then
    echo -e "${RED}❌ Integration tests failed${NC}"
    exit 1
fi

# Executar testes E2E (opcional)
if [ "$1" == "--e2e" ]; then
    echo -e "${YELLOW}🎭 Running E2E tests...${NC}"
    npm run test:e2e
    if [ $? -ne 0 ]; then
        echo -e "${RED}❌ E2E tests failed${NC}"
        exit 1
    fi
fi

echo -e "${GREEN}🎉 All tests passed!${NC}"

    Best Practices¶

    1. Test Organization¶

    • Organize tests by feature/module
    • Use descriptive test names
    • Group related tests with describe blocks
    • Use beforeEach/afterEach for setup/cleanup

    2. Mock Strategy¶

    • Mock external dependencies
    • Use real objects for integration tests
    • Avoid over-mocking
    • Reset mocks between tests

    3. Assertions¶

    • Use specific matchers
    • Test one thing at a time
    • Include negative test cases
    • Test edge cases and error conditions

    4. Test Data¶

    • Use factories for test data creation
    • Avoid hardcoded values
    • Clean up test data after tests
    • Use realistic test data

    5. Performance¶

    • Keep tests fast
    • Run unit tests frequently
    • Parallelize when possible
    • Use appropriate test timeouts
